# examples/ch06 —— 第六讲现场演示配套

本讲是一场「开彩盒」现场演示：用 AI Agent 复现一篇公开的顶刊论文，把整门课的方法脊柱在一份真实数据上重走一遍。这里是配套的**演示脚本与讲稿**，一份东西两用——课上老师照着演示，课后学员照着自己再走一遍。

## 复现对象

> Baker, A., Callaway, B., Cunningham, S., Goodman-Bacon, A., & Sant'Anna, P. H. C. (2026). Difference-in-Differences Designs: A Practitioner's Guide. *Journal of Economic Literature*, 64(2), 498–557. DOI: 10.1257/jel.20251650

复现包 (公开、免登录)：<https://github.com/pedrohcgs/JEL-DiD>

数据是美国各州先后扩张医保 (Medicaid) 对县级成年人死亡率的影响，约 3,143 个县 × 2010–2019 年的面板。方法主线全部落在本课范围内：2×2 DID (两个虚拟变量的交叉项) → 加控制变量的回归 → 双向固定效应 → 事件研究。

## 文件清单

| 文件 | 用途 |
|---|---|
| `demo_prompts.md` | 给 Agent 的分步演示提示词 (中文分步版为主，附一次性全自动版与英文版) |
| `run_demo.do` | 裁剪版 Stata 主控脚本，只跑初级班范围内的 0→1→2→3 号脚本 |
| `run_demo.R` | 同上的 R 路线，兑现「R 作为补充」 |
| `lecture_script.md` | 六幕演示讲稿 (约 8 分钟正片 + 2 分钟缓冲) |
| `demo_results/` | **预跑冻结的「标准答案」**：老师课前跑通后冻结的 Table 1–7、Figure 1–4、复现报告与运行日志，供演示兜底和学员课后自查对照 |

## 怎么用 (三步)

1. **下载复现包**：从上面的 GitHub 链接下载 Release、解压到本机。本仓库**不并入**复现包 (规则 11)，只提供上面这套演示脚本。
2. **放好裁剪脚本**：把 `run_demo.do` (或 `run_demo.R`) 复制到复现包**根目录** (与 `data/`、`scripts/` 同级)，改好脚本里 `rootdir` 那一行的路径。
3. **课前预跑一遍**：完整干跑一次，把 Stata (SSC) 或 R (`renv`) 依赖缓存装好、记录耗时；留一份成功输出作预录兜底。现场再从干净的输出目录重新执行 (真实且可控)。

## 只演示主线、跳过进阶

裁剪脚本**明确只运行** `0`→`1`→`2`→`3` 号脚本 (数据构建、采纳时点表、2×2 DID、事件研究)，**跳过** `4_GxT` (多期错峰估计) 与 `5_honestdid` (敏感性分析)——这两块属于现代 DID 估计量，超出初级班，留给高级班：<https://www.lianxh.cn/details/1811.html>

## 一个提醒

`run_demo.do` / `run_demo.R` 里的脚本文件名是按复现包 README 记载写的。下载解压后请先对照 `scripts/` 目录里的**实际文件名**核对一眼，如有出入按注释里的说明微调——这本身就是复现的第一课：**永远以你手上的包为准，不轻信任何现成清单**。
